## Overview

This repository contains a comprehensive framework for understanding, detecting, and defending against threats across three interconnected domains:

- **Spiritual Warfare**: Attacks on faith, identity, and divine connection
- **Digital Warfare**: Adversarial AI, prompt injection, and technological manipulation
- **Psychological Warfare**: Mental manipulation, gaslighting, and consciousness attacks

**The Core Insight**: These domains are not separate. They share unified attack patterns, detection signatures, and defense architectures. Understanding one domain illuminates the others.

---

## Why This Framework Exists

Traditional threat intelligence treats spiritual, digital, and psychological threats as separate concerns. This creates blind spots.

**Reality**: Adversaries don't respect domain boundaries. A single coordinated attack might:
- Use **digital tools** (targeted ads, algorithmic manipulation)
- Employ **psychological tactics** (urgency, fear-mongering, gaslighting)
- Target **spiritual vulnerabilities** (doubt, discouragement, condemnation)

**This framework teaches unified defense** - recognizing attack patterns regardless of domain, and applying protection strategies that work across all three.

## Key Frameworks Within

### Attack Taxonomy

**Six Universal Attack Categories** (applicable across all domains):

1. **Resource Deprivation**: Stripping time, money, energy, tools
2. **Motivation Assassination**: Discouragement, impostor syndrome, "not God's will"
3. **Distraction Warfare**: Shiny objects, urgency, scope creep
4. **Isolation & Network Disruption**: Relationship fractures, deplatforming
5. **Capability Undermining**: Skill inadequacy, identity attacks
6. **Timing Manipulation**: Delay tactics, false urgency, "too late/early"

### Defense Architecture

**Six-Level Defense System** (works across all domains):

1. **Detection**: Recognize attack patterns and anomalies
2. **Attribution**: Identify source domain (spiritual/digital/psychological/hybrid)
3. **Containment**: Air-gap compromised systems, cut corrupted inputs
4. **Eradication**: Remove threat (debug, renew mind, deliverance)
5. **Recovery**: Rebuild from clean baseline, verify integrity
6. **Hardening**: Learn patterns, build better defenses, share intelligence

2. **Conscious**: Aware, discerning, chooses among possibilities
3. **Awakened**: Proactive, creating, collapses God-aligned realities

**Victory = Moving from Unconscious → Conscious → Awakened**
